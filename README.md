 <p>  O sistema a ser desenvolvido visa melhorar a organização de tarefas com cartões de anotação de fácil manipulação, notificações inteligentes e colaboração escalável e simplificada.

- Criar um quadro de anotações;
- Criar uma anotação;
- Excluir um quadro;
- Visualizar uma anotação;
- Editar uma anotação;
- Excluir uma anotação;
- Pesquisar uma anotação;
- Compartilhar uma anotação;
- Notificar os usuários
